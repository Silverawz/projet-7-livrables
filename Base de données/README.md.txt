Le "script de création de la base de donnée" permet de créer la base de donnée ainsi que les tables vides.

Le "Script d'injection du jeu de données (bibliotheque-microservice)" permet de remplir la base avec les données, pour cela il faut :
-Creer une base de donnée vierge nommée "bibliotheque-microservice" dans Mysql.
-Importer le fichier "Script d'injection du jeu de données (bibliotheque-microservice)" dedans.
